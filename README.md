# School Courses Tutor

Designed to accurately solve relevant science or business problems involving calculations, this approach utilises the natural language understanding capabilities of large language models (LLMs) whilst calling upon pre-written functions to ensure the accuracy of computational results and avoid any potential computational hallucinations that may arise from large models.

## Technical Overview
At this stage, we are conducting tests using on-premises large language models based on Rapid-MLX or Ollama; the current test model is Qwen3.5:4b, which is based on Rapid-MLX. The large language model currently utilises OpenAI’s API to provide its functionality, so it is theoretically compatible with OpenAI or any integration methods compatible with OpenAI.

## Outlook for the implementation of the feature
